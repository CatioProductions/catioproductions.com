# catioproductions.com
future website for catio productions

I'm building a static website for catioproductions.com — a brand site for my indie Windows app business, Catio Production. The site has two missions: selling Windows apps and funding cat rescue/TNR work.
Set up a clean static HTML/CSS project with this structure:

index.html — home/landing page
apps/index.html — app catalog
apps/virtual-screen-manager.html — product page for Virtual Screen Manager
apps/magnadesk.html — coming soon stub for MagnaDesk
rescue/index.html — cat rescue mission + TNR info
rescue/donate.html — donation page (links out to Zeffy/GoFundMe, placeholders for now)
about.html — about the person behind it
css/style.css — single shared stylesheet

Requirements:

No frameworks, no build tools, no npm — plain HTML5 and CSS
Mobile responsive using CSS Grid/Flexbox
Shared nav and footer via repeated HTML for now (no JS includes)
Placeholder text and images are fine — structure and layout are the priority
GitHub Pages compatible (no server-side anything)
